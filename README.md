# Devlogger

### How to run:

- install [git](https://git-scm.com/downloads) and [node](https://nodejs.org)
- open terminal
- run "git commit https://github.com/MaximilianHagelstam/devlogger.git"
- open project folder in terminal
- run "npm install"
- run "node app"
- open "localhost:3000" in browser

### Technology stack:

- Node JS
- Express JS
- Mongo DB
- Bootstrap
